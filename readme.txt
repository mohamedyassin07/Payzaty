PayZaty woocommerce plugin

== Installation ==

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `PayZaty` and click 'Install Now' or click on the `upload` link to upload `payzaty.zip`
4. Click on `Activate plugin`
